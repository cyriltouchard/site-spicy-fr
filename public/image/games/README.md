# 🎮 Images des jeux

Placez ici les images de couverture des jeux streamés.

## Format recommandé

- **Dimensions** : 600x800px (ratio 3:4)
- **Format** : JPG, PNG ou AVIF
- **Poids** : Moins de 200 Ko par image

## Nomenclature des fichiers

Les noms de fichiers doivent correspondre à ceux définis dans `data/games.ts` :

- `warzone.jpg` - Call of Duty: Warzone
- `pubg.jpg` - PUBG: BATTLEGROUNDS
- `gtav.jpg` - Grand Theft Auto V
- `just-chatting.jpg` - Just Chatting
- `bo4.jpg` - Call of Duty: Black Ops 4
- `minecraft.jpg` - Minecraft
- `sea-of-thieves.jpg` - Sea of Thieves
- `scum.jpg` - SCUM
- `special-events.jpg` - SPECIAL EVENTS
- `apex.jpg` - Apex Legends
- `placeholder.jpg` - Image par défaut (optionnel)

## Placeholder automatique

Si une image n'est pas disponible, une icône de manette grise s'affiche automatiquement.

Vous pouvez définir `hasImage: false` dans `games.ts` pour forcer l'affichage du placeholder.
